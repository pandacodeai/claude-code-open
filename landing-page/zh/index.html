Web UI 全栈 IDE

基于 React + Express + WebSocket 的完整 Web 界面。代码浏览器、终端模拟、实时消息流、OAuth 认证、设置面板 —— 在浏览器中获得完整的 AI 编程体验

Agent Teams 多智能体

创建 AI 团队进行协作开发。支持团队创建、成员管理、智能体间消息通信、共享上下文记忆，让多个 Agent 并行解决复杂问题

Blueprint 工作流引擎

可视化设计多智能体工作流。支持蓝图生成、Lead Agent 调度、Worker 分派、任务计划更新、端到端测试触发，实现复杂任务自动编排

35+ 内置工具

智能上下文管理

代码深度分析

配置 API Key
